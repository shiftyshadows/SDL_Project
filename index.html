Key Features

Feature 1: Dynamic Raycasting

Our project uses real-time raycasting to dynamically render walls and objects.

Feature 2: Smooth Player Movement

Players can move and rotate smoothly with full collision detection.

Feature 3: Custom Map Parsing

Load custom maps from external files, allowing for unique user-created environments.

About the Project

This project was inspired by the classic raycasting engines used in early 3D games. I wanted to create something that combined my passion for game development with modern technology. This project is part of my portfolio for Holberton School, where I have been learning full-stack development.
